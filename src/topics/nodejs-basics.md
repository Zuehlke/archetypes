# Node.js Basics

Node.js is a runtime environment that allows developers to execute JavaScript on the server side. This topic introduces the fundamentals of Node.js, including its architecture, core modules, and basic usage.

## Key Concepts

- **Event-Driven Architecture**: Understanding the non-blocking I/O model.
- **Core Modules**: Using modules like `fs`, `http`, and `path`.
- **Package Management**: Managing dependencies with npm or Yarn.
- **Creating a Simple Server**: Building a basic HTTP server.
- **Asynchronous Programming**: Using callbacks, promises, and async/await.
